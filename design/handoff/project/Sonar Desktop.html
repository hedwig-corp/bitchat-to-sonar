Sonar Desktop

macOS / iPadOS split view · same design system as mobile

Layout

  • Sidebar — status chip, channels & messages, you + settings at the bottom.
  • Main pane — chat with the same transport-colored bubbles (cyan = Bluetooth, indigo = internet).
  • Detail rail — safety number & verify for DMs; place + precision for channels. Toggle with the ⓘ button.
  • Radar gets a full pane with the peer list beside it.

Try

  • Click Sonar at the top of the sidebar → radar pane; click a peer to chat.
  • Maya (cyan) vs Sofia (indigo) — watch the composer send button change too.
  • ⓘ in the chat header → verify Maya's safety number in the rail.
  • Open a channel → rail shows the place precision picker.
  • Bottom-left gear-ish button → Settings modal (rename yourself, wipe, toggles).
  • Tweaks: window width simulates iPad (~1080px) vs full desktop.

iPad note

  • This three-pane layout maps 1:1 to iPadOS: sidebar = primary column, rail = inspector; at iPad width the rail overlays instead.
