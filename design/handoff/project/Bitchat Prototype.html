bitchat — interactive prototype

v1 · core flow · light & dark · tweakable

Design assumptions

  • Familiar Signal/Telegram patterns: conversation list, bubbles, a real composer — zero terminal idioms required.
  • Amber #D97706 accent on warm-neutral surfaces; true-neutral dark mode.
  • Figtree for UI; IBM Plex Mono only for fingerprints and keys.
  • Network state is simulated for the demo.

Try

  • Tap the status chip to flip Online ↔ Offline (mesh) — then send a message and check its delivery tag.
  • Open Lugano · Centro — public banner, colored authors, mesh/internet tags per message.
  • DM Maya → Verify → compare safety numbers → mark verified.
  • Type / in any composer for the power-user command layer.
  • People icon (top right) → mesh peers in plain language.

3 directions — open Tweaks

  • quiet — iOS-classic, hairline lists
  • warm — grouped cards on warm canvas
  • soft — airy, no separators

Plus chip/bubble variants, radius, density, and typefaces.
